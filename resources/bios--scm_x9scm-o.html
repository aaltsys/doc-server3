SuperMicro MBD-X9SCM-O BIOS¶

First BIOS Boot¶

  1. Boot the system, pressing <Del> multiple times to enter the BIOS setup.
  2. At Exit ‣ Restore Defaults ‣ Load Optomized Defaults on the menu, press <Enter> for Yes.
  3. At Advanced ‣ Boot Feature ‣ Power Button Function on the menu, press <Space> to select 4 Seconds Override.
  4. At Advanced ‣ IDE/SATA Configuration,
    1. At SATA Mode, press <Space> to select RAID Mode.
    2. At :menuselection: PCH RAID Codebase, press <Space> to select Intel.
    3. At Serial ATA Port 0 ‣ Hot Plug, press <Space> to select Disabled (should show first RAID drive).
    4. At Serial ATA Port 0 ‣ Hot Plug, press <Space> to select Disabled (should show second RAID drive).
    5. At Serial ATA Port 0 ‣ Hot Plug, press <Space> to select Disabled (should show DVD-RW drive).
  5. At Exit ‣ Save Changes and Reset, press <Enter> to save changes, and answer the question Save configuration and reset? with Yes.

Second BIOS Boot¶

  1. Press <Ctrl-I> multiple times to enter the Intel RAID utility.
  2. Press 3 for Reset Disks to non-RAID. Press <Space> to select disks, then press <Enter> to finish the operation.
  3. Press 1 for Create RAID volume.
    1. At Name:, enter aaltsys <Enter>.
    2. At RAID Level:, press <Down> to display RAID1(Mirror).
    3. At Capacity, press <Enter> to select the entire drive.
    4. Respond to the Create Volume prompt with <Enter>, and confirm the Are you sure ...? question with Y.
  4. Press 6 to Exit the RAID utility, and confirm the Are you sure ...? question with Y.

Third BIOS Boot¶

  1. Boot the system, pressing <Del> multiple times to enter the BIOS setup.
  2. At Advanced ‣ IDE/SATA Configuration ‣ SATA Mode, press <Space> to select AHCI Mode.
  3. At Exit ‣ Save Changes and Reset, press <Enter> to save changes, and answer the question Save configuration and reset? with Yes.

Fourth BIOS Boot¶

  1. Boot the system, pressing <Del> multiple times to enter the BIOS setup.
  2. At Boot ‣ Boot Option Priorities ‣ Boot Option #1, press <Enter> and then select P2: HL-DT-ST DVDRAM ....
  3. At Boot Option #2, select P0: WDC WD5000 ....
  4. At Exit ‣ Save Changes and Reset, press <Enter> to save changes, and answer the question Save configuration and reset? with Yes.

The system will now boot from the DVD drive, and install an OS on the RAID drive set.
