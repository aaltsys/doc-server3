Note

BIOS Notes

  • BIOS Features, SuperBoot – set Disabled so logo screen will display
  • BIOS features, SuperBIOS Protect – set Enabled to prevent BIOS corruption
  • Advanced BIOS Features, Delay Prior to Thermal – set to the minimum interval for a complete boot
  • Integrated Peripherals, SATA Mode – set to RAID
  • Integrated Peripherals, SATA Port Speed Settings – Force GEN II for highest data transfer rates
  • Integrated Peripherals, SiI 3114 SATA Controller – RAID Mode for disk mirroring
  • Power Management Setup, ACPI Function – Enabled allows soft power off on shutdown

Flash Foxconn 955X7AA BIOS¶

The Foxconn MB is capable of being flashed with a custom BIOS and boot splash, appropriate for OEM distribution. Use these notes to install the custom BIOS.

Warning

Instructions in this article apply to internal services at AAltsys Technology and are not intended for general use.

Enable BIOS Flash Settings¶

  1. Insert the bootable flash drive in a USB port
  2. Turn on the CPU, press < Del > to enter setup when prompted
  3. Choose Advanced BIOS Features from the CMOS Setup Utility #. At Hard Disk Boot Priority, press <Enter> #. Highlight 'USB-HDD0‘, press + until it is at the top of the list #. Press <Esc> to return to the main menu
  4. Choose BIOS Features #. Set SuperBIOS-Protect to 'Disabled‘
  5. Press <F10>, then Y to Save and Exit

Boot From USB Flash Drive¶

  1. Boot from the USB Drive, starting BartPE...
  2. When asked, “Do you want to start network support now?”, answer No
  3. On the manu, select Go + Programs + SuperLogo

Flash the Startup Logo¶

  1. Click the Change Logo pushbutton
  2. Click the Full Screen radiobutton
  3. Click Next
  4. At “Logo File,” Select Browse...
  5. Show files of type Jpeg Files (*.jpg)
  6. Click “Look in:”, select drive WARESPE (X:)
  7. Open folder Boot Images
  8. Select file BootLogo.jpg
  9. Click Next
  10. Click button Update
  11. At “Do you really want to change logo?”, choose Yes
  12. At the message, “Can not open file flash.txt for reading,” click OK
  13. At the message, “Unable to get flash information!,” click OK
  14. WAIT FOR THE MESSAGE, “You must restart your system in order for the new logo to take effect!,”
  15. Close the SuperLogo program, then restart the computer.

Restart the system¶

  1. Remove the USB Flash drive
  2. Press < Del > to enter the CMOS Setup Utility
  3. Select BIOS Features, change “SuperBIOS-Protect” to Enabled
  4. Press <F10>, then Y to save the new settings
